## 一、产品定位

**一句话：** AI 视频制作工具的"天眼查"——把散落在各处的价格、功能、评测、评分整合到一个页面，帮用户快速决策。

**我们不做：**
- ❌ 自己评测工具
- ❌ 自己生成对比视频
- ❌ 自己写长篇技术博客

**我们做：**
- ✅ 采集官方定价 → 结构化对比
- ✅ 采集官方展示视频 → 嵌入工具页
- ✅ 采集第三方评测视频 → 嵌入对比页
- ✅ 采集 G2/社区评分 → 展示口碑
- ✅ 自动生成 500+ 对比页面

## 三、数据采集方案

### 数据源矩阵

| 数据类型 | 来源 | 采集方式 | 自动化程度 |
|:--------|:-----|:--------|:---------:|
| 定价/功能 | 各工具官网 | requests / Playwright | ✅ 全自动 |
| 官方展示视频 | YouTube API | 搜索官方频道 + 筛选 | ✅ 全自动（选一次固定） |
| 第三方评测视频 | YouTube API | 按播放量/时间筛选候选，人工确认 | ⚡ 半自动（选一次管半年） |
| G2 评分 | Google 搜索摘要 | 提取搜索结果中的评分 | ✅ 全自动 |
| Reddit 口碑 | agent-reach / tinyfish | 搜索 Reddit 讨论，提取摘要 | ✅ 全自动 |
| 新功能/更新 | 各工具 Blog / Changelog | RSS / 网页抓取 | ✅ 全自动（检出变化通知） |

### 更新频率

| 数据 | 频率 | 说明 |
|:----|:----|:-----|
| 定价 | 每月1次 | 这些工具半年到一年才调一次价 |
| 评测视频 | 选一次固定 | 除非有大版本更新才重新选 |
| G2评分 | 每月1次 | 评分变化慢 |
| Reddit口碑 | 每月1次 | 聚合最新讨论 |
| 新工具加入 | 按需 | 发现值得加的新工具就加 |

## 六、核心优势总结

```
竞品（大媒体综述）           我们
────────────────────────────────────
一年更新一次                 每月更新定价
手工写一篇文章                程序化生成500+页面
不带定价对比表               结构化定价数据
靠编辑判断"哪个最好"          靠数据展示"哪个适合你"
没精力做所有工具对比         加个工具 = JSON加一行
```

> 我们不拼内容深度，拼**覆盖面 × 更新频率 × 结构化程度**。这才是程序化SEO的本质。
