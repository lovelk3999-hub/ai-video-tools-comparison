# ROADMAP - AI Video Tools Comparison 长期计划

> 创建日期：2026-06-19
> 目标：建成AI视频工具领域流量最大的对比站，通过广告盈利

---

## 工作原则

1. 能自动化就自动化 — 采集/更新/部署全部自动
2. 不做主观评测 — 只聚合公开数据（官网定价/G2评分/Reddit讨论/官方视频）
3. 遇到blocked先记日志再跳过 — 不卡死在单个任务上
4. 每个完成的任务写入 done/ 并更新 STATUS.md
5. 每次重大决策追加写入 DECISIONS.md

---

## 遇到无法解决的问题 → 应对策略

| 场景 | 策略 |
|------|------|
| CAPTCHA/反爬 | 换用TinyFish API或Playwright真人浏览器模式，若仍失败则跳过，记录到DECISIONS.md |
| 某工具官网改版导致定价抓取失败 | 跳过该工具，标记为"需人工检查"，继续其他工具 |
| GitHub Actions部署失败 | 本地手动构建+推送重试，连续3次失败则切到本地部署 |
| 需要你决策（域名/付费服务/账号注册） | 写 doing/ 任务标记为 [NEEDS_USER]，等你回来处理 |
| Google AdSense申请被拒 | 记录拒绝原因，补充内容后重新申请 |
| 新工具出现但信息不足 | 只收录有明确定价页面的工具，信息不全的先跳过 |
| 数据源（G2/Reddit）API变更 | 改用Firecrawl或手动兜底方案 |
| Search Console收录异常 | 检查robots.txt和sitemap，排除封禁可能 |
| 定价页面需要登录才能看 | 使用已登录的浏览器会话采集，或标记为需手动更新 |
| YouTube视频链接失效 | 搜索替代视频，若无则删除该条目 |
