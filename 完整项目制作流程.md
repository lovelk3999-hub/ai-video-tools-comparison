## 1. 项目定位

**AI视频工具定价对比站** — 聚合AI视频生成工具的定价、功能、评分数据，做中立对比。

- 不生产原创评测内容
- 只聚合公开数据（官网定价、G2评分、Reddit讨论、官方视频）
- 盈利方式：Google AdSense广告收入
- 目标流量来源：Google搜索（SEO）

## 9. 编码注意事项（Windows环境）

1. **文件I/O**：必须 `encoding="utf-8"`，否则中文乱码
2. **print(emoji)**：Windows GBK终端会崩溃，用文字替代
3. **PowerShell &&**：不支持，用 `;` 或分条执行
4. **CSS {}**：在Python f-string中要用 `{{}}` 转义
5. **PowerShell别名**：`curl` 是 `Invoke-WebRequest` 的别名，用 `curl.exe`

## 10. 故障应对策略

| 问题 | 对策 |
|------|------|
| CAPTCHA/反爬 | TinyFish API / Playwright真人模式，不行就跳过 |
| 工具官网改版 | 标记"需人工检查"，继续其他 |
| GitHub Actions失败 | 本地重试，3次不行切本地部署 |
| 需要用户决策 | 标记 [NEEDS_USER]，等用户处理 |
| AdSense被拒 | 记录原因，补充内容再申 |
| 视频链接失效 | 搜替代品，没有再删 |
| Search Console异常 | 检查robots.txt和sitemap |
